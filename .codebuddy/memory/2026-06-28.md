# 2026-06-28

## JMCL DMG 下载修复

**问题**：JMCL 项目的 `JMCL-2026.3.0.dmg`（125MB）无法正确下载。

**根因**：GitHub Pages **不支持 Git LFS**。DMG 文件被 LFS 管理后，GitHub Pages 上托管的是 LFS 指针文件（~130 字节文本），而不是真正的二进制文件。用户下载到的是指针文本，不是 DMG。

**修复**：将 `file/JMCL/main.json` 中下载 URL 从相对路径 `../file/JMCL/JMCL-2026.3.0.dmg` 改为 GitHub raw URL `https://raw.githubusercontent.com/Open-code-Studio/downloads/main/file/JMCL/JMCL-2026.3.0.dmg`。raw.githubusercontent.com 会正确处理 LFS 文件，返回真实二进制。

**其他**：
- 项目使用 `*.dmg filter=lfs` 追踪大文件
- GitHub 推送时遇到 LFS lock verify 超时，需要 `git config lfs.locksverify false`
