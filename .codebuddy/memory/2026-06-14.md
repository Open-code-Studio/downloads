# 2026-06-14

## OCS 下载站 - 修复"没有找到项目"空状态问题

**问题**：页面显示"没有找到项目 尝试调整搜索条件或筛选器"，项目卡片区域为空。

**根因**：`page/config.json` 中 `projects` 配置为 `["publish1"]`，但实际项目文件夹是 `file/Open-web-download.kit 1.0/`，导致 `main.js` 中 `fetch("../file/publish1/main.json")` 404，项目列表为空。

**修复**：
1. 更新 `config.json` 的 `projects` 数组：`"publish1"` → `"Open-web-download.kit 1.0"`
2. 修复 `main.json` 中的文件下载 URL：原来引用不存在的 `.zip` 文件，改为实际存在的 `example-app-linux-x64.AppImage`

**项目结构**：
- 入口：`index.html`（3秒自动跳转到 `page/index.html`）
- 主页面：`page/index.html`（搜索 + 项目卡片 + 详情面板）
- JS：`page/js/config.js`（加载配置）、`page/js/main.js`（主逻辑）
- CSS：`page/css/md3.css`（设计系统）、`page/css/main.css`（附加样式）
- 数据文件：`file/{项目名}/main.json`
